1) BOLHA - Bubble Sort
Estável
Memória Interna
Alta complexidade (muito esforço)
A estrutura possui 2 partes/porções (1a é a desordenada; 2a parte é a ordenada)
Adequado para vetores e listas

Trabalha com o conceito de trocas, ou seja, enquanto houver trocas, o algoritmo não para.
É considerado bolha, porque 'leva' os maiores para o final da estrutura

void bolha(int *v, int n) {
    int i;
    int houveTroca;
    int tmp;

    do {
        houveTroca = 0;
        for (i = 0; i < n-1; i++){
            if (v[i] > v[i + 1]) {
                houveTroca = 1;
                tmp = v[i];
                v[i] = v[i+1];
                v[i+1] = tmp;
            }
        }
    } while (houveTroca);
}

Celula *bolha(Celula *l) {
    Celula *i;
    int houveTroca;
    int tmp;

    if (!l) return l;

    do {
        for (i = l; i->prox; i = i->prox){
            if (i->dado > i->prox->dado) {
                houveTroca = 1;
                tmp = i->dado;
                i->dado = i->prox->dado;
                i->prox->dado = tmp;
            }
        }
    } while (houveTroca);
}
